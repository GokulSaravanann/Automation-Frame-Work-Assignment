
# Test Automation Framework

This is a java based test automation framework.

It is designed to provide robust, scalable and maintainable testing for web applications.

The framework leverages various libraries and tools to facilitate data-driven testing, logging, reporting and integration with cloud-based testing platforms like lambda test.
